This repository  includes automated tests for the `https://www.saucedemo.com` website.


## Packages
The project uses the following packages:
* Selenium WebDriver 3.141.59
* ChromeDriver 83.0.4103.97 
* Java JDK 11 or higher
* Apache Maven 3.6.3
* TestNG 7.1.0
* Cucumber 6.2.2
